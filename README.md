## API Reference

#### API used

Link: http://universities.hipolabs.com/search?name=korea

#### Get name


| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `university.name` | `string` | Name of universities |

#### Get website


| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `university.web_pages[0]`      | `string` | Link to university website |
